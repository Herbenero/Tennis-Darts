Tennis Darts

Alternate sides, lock a target, win service games, survive deuce, and take the set.

Setup Mode
Rally scoring

Serve, return, and break the board.

Each service game begins on the left side of the board: 5, 12, 9, 14, 11, 8, 16, 7, 19. The next point alternates to the right side: 1, 18, 4, 13, 6, 10, 15, 2, 17. Numbers 20 and 3 are the net and lose the point. The server’s first valid hit sets the Target Number. A double on the first dart is an Ace and instantly wins the point. After that, the server keeps hitting the target to build a score. Once the server misses the target, play passes to the returner, who must beat the server’s high score on the Target Number. First to win 4 service games wins the set, with tennis scoring 15, 30, 40, game. At 40-40, use a tiebreak. If the set reaches 3-3, play a set tiebreak where the server must win two consecutive points.

Active side Only the current side’s numbers are valid. If the first two darts miss the active side, the point is lost.
Target rally The server establishes a target number, then tries to stack score on that same number until missing it.
Tennis scoring Points progress 15 → 30 → 40 → game. Deuce uses a tiebreak. A 3-3 set uses a special set tiebreak.
